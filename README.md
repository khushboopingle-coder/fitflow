# 🏋️ FitFlow – Fitness App UI Prototype

A mobile fitness app UI prototype built as part of my **UI/UX Internship at SkillCraft Technology** (March 2026).

---

## 📱 Live Preview

Open `fitflow.html` in any browser — no installation or setup required!

---

## ✨ Features

- 🚀 **Splash Screen** – App intro with feature highlights
- 🎠 **Onboarding Flow** – 3-step walkthrough carousel
- 📝 **Create Account** – Sign-up form with Google & Apple options
- 👤 **About You** – Age, weight, height & gender setup
- 🎯 **Goal Selection** – Lose Weight, Build Muscle, Stay Active, Run Faster
- 🏠 **Home Dashboard** – Personalized greeting, calorie counter & streak tracker
- 💪 **Workout Screen** – Full Body Intro with 5 exercises, tap to complete
- 📊 **Stats Screen** – Weekly workout bar chart, streak & calorie summary
- ⚙️ **Profile Screen** – User profile and settings menu

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- Google Fonts (Sora + DM Sans)

> Zero dependencies — everything is in a single file!
